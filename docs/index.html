תיעוד MyDan

ברוכים הבאים לתיעוד המלא של פלטפורמת MyDan. כאן תמצאו את כל המידע הנדרש כדי להתחיל ולהשתמש במערכת בצורה יעילה.

התחלה מהירה

MyDan היא פלטפורמה מתקדמת לניהול פרויקטים המאפשרת לכם לנהל פרויקטים, משתמשים ותשלומים במקום אחד.

טיפ: לפני שתתחילו, ודאו שיש לכם חשבון פעיל במערכת.

שלבי התחלה

  1. היכנסו למערכת באמצעות פרטי ההתחברות שלכם
  2. עברו ללוח הבקרה כדי לראות סקירה כללית
  3. צרו פרויקט חדש או הצטרפו לפרויקט קיים
  4. הגדירו את ההרשאות והתפקידים של הצוות

לוח בקרה

לוח הבקרה הוא המרכז של הפעילות שלכם במערכת. כאן תוכלו לראות:

  • סטטיסטיקות כלליות
  • פרויקטים פעילים
  • פעילות אחרונה
  • תשלומים ממתינים
// דוגמה לנתוני לוח בקרה { "total_projects": 12, "active_projects": 8, "completed_tasks": 156, "pending_payments": 3 }

ניהול פרויקטים

ניהול פרויקטים ב-MyDan מאפשר לכם ליצור, לערוך ולעקוב אחר פרויקטים בצורה יעילה.

יצירת פרויקט חדש

  1. לחצו על "פרויקט חדש" בלוח הבקרה
  2. מלאו את פרטי הפרויקט (שם, תיאור, תאריך יעד)
  3. הוסיפו חברי צוות לפרויקט
  4. הגדירו משימות ומטרות
חשוב: ודאו שכל חברי הצוות קיבלו הרשאות מתאימות לפרויקט.

ניהול תשלומים

מערכת התשלומים מאפשרת לכם לנהל את כל התשלומים והחשבוניות במקום אחד.

תכונות עיקריות

  • יצירת חשבוניות אוטומטית
  • מעקב אחר תשלומים
  • דוחות פיננסיים
  • אינטגרציה עם מערכות תשלום

ניהול משתמשים

ניהול משתמשים מאפשר לכם לשלוט בהרשאות ובגישה למערכת.

סוגי משתמשים

  • מנהל מערכת: גישה מלאה לכל הפונקציות
  • מנהל פרויקט: ניהול פרויקטים ספציפיים
  • חבר צוות: גישה מוגבלת לפרויקטים מוקצים
  • לקוח: צפייה בלבד בפרויקטים רלוונטיים

API

MyDan מספקת API מלא לשילוב עם מערכות חיצוניות.

// דוגמה לבקשה API GET /api/v1/projects Authorization: Bearer YOUR_API_KEY Response: { "projects": [ { "id": 1, "name": "פרויקט לדוגמה", "status": "active" } ] }

שאלות נפוצות

איך אוכל לשנות סיסמה?

לכו להגדרות החשבון ולחצו על "שנה סיסמה". תצטרכו להזין את הסיסמה הנוכחית ואת הסיסמה החדשה.

איך אוכל להוסיף משתמש חדש?

מנהלי מערכת יכולים להוסיף משתמשים חדשים דרך דף "ניהול משתמשים" בלוח הבקרה.

איך אוכל לייצא דוחות?

בכל דף דוחות יש כפתור "ייצא" שמאפשר להוריד את הנתונים בפורמטים שונים (PDF, Excel, CSV).
